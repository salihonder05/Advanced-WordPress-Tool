# Advanced-WordPress-Tool
Manage Your WordPress Settings For First Installation
(İlk Kurulumda wordpress için ayarlarınızı yönetin.)

# Features
  Disable XMLRPC
  Disable WP Emojis
  Disable WP Version Number
  Disable Manifest
  Disable Shortlink
  Disable Query Strings All Static Resources
  Disable api.w.org Relation Link
  Disable Rest API for Guests (site.com/wp-json)
  Disable Dns Prefetch Link
  Disable WP Embed Link
  Disable RSS Links
  Disable Gutenberg Editor
  Disable Theme/Plugin File Editors
  Disable Html In Comments (For Safe Comments)
  Disable Comment System	You must remove frontend components.
  Disable Admin Bar From Frontend
  Disable Auto Save (In Edit Screens)
  Add SVG Support (Media - Upload)
  Disable Screen Options Button (Administrators Excluded)
  Disable Previous and Next Article Links From Head (Html)
  Move Jquery To Google CDN From Wp-Includes
  Disable WP Core Auto Update
  Disable Themes Auto Update
  Disable Plugins Auto Update

  Add Scripts To Header

# Özellikler
  XMLRPC' yi Header'dan kaldır
  WP Emojilerini İptal Et
  Wordpress versiyon numarasını header'dan kaldır
  Manifest Linkini Kaldır
  Shortlink'i Header'dan kaldır
  Statik Kaynaklardan Sorgu Dizelerini Kaldır
  api.w.org Relation Linkini Kaldır
  Ziyaretçilerin REST API erişimini kaldır (site.com/wp-json)
  DNS Prefetch Linkini Kaldır
  WP Embed Linkini iptal et.
  RSS Linklerini Header'dan Kaldır
  Gutenberg Editörünü Kaldır
  Tema/Eklenti Düzenleyiciyi İptal Et
  Yorumları HTML den temizle
  Yorum Sistemini Kaldır	Temanızda kalıntıları temizlemeniz gerekir.
  Admin Bar'ı siteden kaldır (Admin panelden kaldırmaz)
  Otomatik Kaydetmeyi Kapat (Düzenleme ekranlarında)
  SVG desteği ekle (Medya dosyaları için Upload)
  Ekran Tercihleri Butonunu Admin Panelden Kaldır. (Adminler Hariç)
  Önceki ve Sonraki Yazı Bağlantılarını Header'dan Kaldır
  WordPress Jquery kaynağını CDN e taşı
  Wordpress'in Otomatik Güncellenmesini Engelle
  Temaların Otomatik Güncellenmesini Engelle
  Eklentilerin Otomatik Güncellenmesini Engelle

  Header'a script Ekleme
